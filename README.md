# env
My personal environment variables

## Prerequisites

- Zsh
- [Prezto](https://github.com/sorin-ionescu/prezto)
- [Homebrew (Mac)](https://brew.sh/)

## Installation

### Linux

```bash
sudo apt install zsh
```

### Mac

```bash
brew install zsh zsh-completions
```

#### Set Zsh as default shell

```console
chsh -s /bin/zsh
```
